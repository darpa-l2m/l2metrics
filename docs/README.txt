To compile a new version of this pdf, "metrics_documentation.pdf" on a Linux system, you must:

1) Download and install texlive-full. On Ubuntu, simply `apt-get install texlive-full` - else see  https://www.tug.org/texlive/quickinstall.html for OS-specific instructions

2) Run `make` from the ./l2metrics/docs/document_source directory

3) A newly compiled metrics_documentation.pdf will replace any previous version in ./l2metrics/docs directory
